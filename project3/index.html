About, dropdropdrop

라이프스타일 브랜드 ‘드롭드롭드롭(dropdropdrop)’은
디자이너의 감성적인 드로잉과 패턴(Pattern)을 통해 유쾌한 에너지를 주는 브랜드입니다.

‘패턴을 가지고 놀다’ 라는 의미의 ‘패턴 플레이: PATTERN PLAY’를 타이틀로
재기 발랄하고 독창적인 다양한 패턴을 선보이고 있습니다.

Overview

드롭드롭드롭(dropdropdrop)은 유니크하고 위트 있는 디자인으로, 평소 애정을 가지고 있는 브랜드입니다.
이번 프로젝트는 단순한 상품 홍보를 넘어, 브랜드의 아이덴티티와 함께 상품을 효과적으로 전달하는 데 중점을 두었습니다.

브랜드의 유니크한 패턴, 풍부한 컬러감, 그리고 동적인 요소를 활용해 브랜드 아이덴티티를 효과적으로 전달하고,
상품의 매력을 돋보이게 하는 프로모션 페이지를 기획했습니다.

Design Strategy

‘봄의 생동감을, DropDropDrop과 함께 느껴보세요!’

  • • Spring 시즌을 컨셉으로, Bright & Vivid한 컬러 팔레트로 화사하고 활기찬 분위기 구현.
  • • 패턴 시리즈별로 상품을 구성해 소비자가 제품의 특성과 매력을 쉽게 이해하고 즐길 수 있도록 설계.
  • • 동적 애니메이션으로 시그니처 패턴을 활용해 생동감 있는 분위기 연출.

Publishing

  • GSAP를 활용한 텍스트 애니메이션 구현
  • Lottie.js를 활용한 동적 애니메이션 배경 구현
  • Swiper.js로 자동 이미지 슬라이드 쇼 구현
  • GIF 애니메이션을 활용한 자동 이미지 전환 효과 구현
  • :root 변수를 정의하여 재사용성, 유지보수성 개선
  • CSS 애니메이션과 @keyframes 을 사용한 회전 및 흔들림 효과 구현
  • JavaScript를 이용한 커스텀 마우스 커서 구현

Review

이번 프로젝트는 프로모션 콘텐츠의 기획, 디자인, 퍼블리싱을 중심으로 진행되었으며,
단순한 홍보 페이지에서 출발해 브랜드 매력을 효과적으로 전달하는 방향으로 발전시켰습니다.

특히, GSAP와 Lottie.js를 활용해 드롭드롭드롭의 시그니처 패턴에 애니메이션 효과를 구현하는 과정이 흥미로웠고고,
덕분에 프로모션 페이지가 더욱 생동감 있고 유니크한 매력을 갖추게 되었습니다.

동적 기능 구현 과정에서 어려움도 있었지만, 브랜드의 정체성을 잘 반영한 완성도 높은 디자인을 구현할 수 있어 매우 만족스러웠습니다.
이번 프로젝트 경험을 바탕으로, 앞으로도 다양한 웹 기술을 활용해 고객의 시선을 사로잡는 프로모션 페이지를 만들어가겠습니다.
